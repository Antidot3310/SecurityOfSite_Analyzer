# Scanner — прототип сканера уязвимостей веб-форм (MVP)

Кратко
------
Минималистичный прототип для курсовой: извлекает формы из HTML и содержит базовые утилиты для работы с HTTP и локальными файлами. 
Проект имеет базовый функционал. Средняя стадия.

## Текущее состояние
- Flask API `/api/parse?url=...`, `/api/scan?url=` реализованы.
- Парсер форм (включая `input`, `textarea`, `select`) и классы реализованы.
- SQLite хранилище с таблицей `scans` для результатов.
- Базовые unit/integration тесты (pytest).
